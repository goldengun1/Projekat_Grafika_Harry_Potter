# Harry Potter and the OpenGL

University project for Computer Graphics using OpenGL, GLFW, GLM and Glad.

# Controls

**WASD** - camera movement

**MOUSE SCROLL** - zoom in/zoom out

**LSHIFT** - increase speed / blur scene

**C** - creative mode

**L** - _lumos_

**M** - stop the snitch
